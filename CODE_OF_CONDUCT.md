# Code Of Conduct

## Our Standard

This project expects respectful, direct, evidence-based collaboration.

Unacceptable behavior includes harassment, abuse, bad-faith disruption, and knowingly misrepresenting technical evidence.

## Reporting

Report conduct issues to `support@zer0pa.com` or `security@zer0pa.com` when the issue overlaps with a security concern.

## Enforcement

Repository maintainers may remove content, restrict participation, or deny contribution access when conduct degrades the integrity of the project or its collaborators.
