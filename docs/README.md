# Documentation Index

This directory is the front door for the ZPE-IoT documentation surface.

Start here before reading historical runbooks or release bundles.

## Core Docs

- `docs/BENCHMARKS.md`: current evidence-labeled benchmark surface
- `docs/CLI_CONTRACT.md`: CLI behavior and command contract
- `docs/CHEMOSENSE_EXTENSION.md`: chemosense scope and packetization notes
- `docs/FIDELITY_SEMANTICS.md`: fidelity metric definitions and policy
- `docs/TEST_MATRIX.md`: active test surface map
- `docs/RELEASE_CHECKLIST.md`: current release gate truth

## Proof And Audit Routing
