# Public Benchmarks

This directory holds dataset-level JSON receipts derived from the latest public
benchmark surface. These files are supplementary evidence packaging, not a new
front-door authority claim by themselves.

- Source benchmark summaries remain under `validation/results/`
- Dataset provenance remains under `validation/datasets/manifest.json`
- Current exported receipts are generated by `validation/benchmarks/export_public_benchmarks.py`
- `INDEX.json` is the entry point for the bundle and carries the `11 READY + 1 BLOCKED`
  boundary plus the local-fallback observability status from the latest tracking context
