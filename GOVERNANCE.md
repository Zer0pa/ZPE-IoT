# Governance

ZPE-IoT is currently owner-directed.

## Current Decision Rules

- code, tests, manifests, and reproducible artifacts outrank prose
- mixed evidence is not a pass
- release claims require current evidence, not historical success
- package publication and public-facing changes require explicit owner approval

## Current Maintainer Scope

- repository structure and code
- validation and proof surfaces
- release gating and documentation integrity

## Current Non-Goals

- public-release theater before Phase 5 verification
- broad suite-coupling claims without current proof
