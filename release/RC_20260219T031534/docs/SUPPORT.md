# Support Policy

## Channels

- Technical support: `support@zer0pa.com`
- Security reports: see `SECURITY.md`

## Scope

Support covers:

- API/CLI usage issues,
- integration guidance for Rust/Python/C surfaces,
- benchmark reproduction assistance,
- bug triage with minimal reproducible examples.

## Required Ticket Information

- product version or commit SHA,
- runtime environment (OS, Python, Rust toolchain),
- command/API call used,
- logs and stack traces,
- sample input (redacted if needed).

## SLA Targets

- Acknowledge support request within 2 business days.
- Provide triage status within 5 business days.
